## 📌 项目目的

做一个**零维护**的职场权益工具箱网站，靠长尾 SEO 获取自然流量，通过 AdSense + Affiliate 自动变现。

## 🔧 三大核心工具

| 工具 | 功能 | 目标关键词 |
|------|------|-----------|
| **病假权利计算器** | 选国家/州 → 查法定病假天数+工资比例 | sick leave calculator (12K) |
| **辞职信生成器** | 填信息 → 一键生成专业辞职信 | resignation letter generator (45K) |
| **员工权益检测器** | 答问卷 → 判断权益是否被侵犯 | toxic workplace quiz (5K) |

## 💰 变现渠道

1. **Google AdSense** — 全自动广告展示 (~50%)
2. **Indeed/ZipRecruiter CPA** — 求职注册佣金 (~20%)
3. **律师咨询 Affiliate** (~15%)
4. **Pro 辞职信包 ($4.99)** — 数字产品自动交付 (~15%)
